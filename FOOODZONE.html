📋 Product Requirements Document

Product Name

Order Review Web App

Purpose

Enable users to view, modify, and finalize their order stored in the browser's localStorage with an intuitive and responsive interface.

✨ Core Features

🍽️ Display Order Items

  • Fetch cart items from localStorage under key "cart"
  • Show each item with emoji, name, price, and optional ingredients
  • Responsive grid layout (1 to 3 columns)
  • Hover effects reveal additional information

🗑️ Clear Order Button

  • Button labeled "Clear Order"
  • Clears localStorage "cart" and reloads page
  • Shows message if cart is empty
  • Instant UI update

✅ Finish Order Button

  • Calculate total price from cart items
  • Replace container with success message
  • Clear the cart in localStorage
  • Provide "Back to Menu" link

📱 Responsive Design

  • Tailwind CSS for responsive UI
  • Readable text and clickable buttons
  • Custom cursor for unique UX
  • Cross-device compatibility

💬 User Feedback

  • "Your cart is empty" message
  • Hover overlays with item details
  • Success confirmations
  • Clear visual indicators

🔄 User Flow

1
User opens the order review page
2
App loads and displays cart items or empty message
3
User can navigate back to menu, clear order, or finish order
4
After finishing order, user sees total price and confirmation
5
User can return to menu to place new orders
