📘 FoodZone Project Documentation

📝 Project Overview

The FoodZone app is a client-side restaurant ordering interface built with plain HTML, JavaScript, and Tailwind CSS. Users can browse a menu, add items to a cart, and view a live order summary, all within the browser using localStorage.

🛠 Tools & Technologies Used

  • Tailwind CSS
  • localStorage API
  • HTML5 & JavaScript

📚 Learning Journey

Learned how to use Tailwind utilities for layout and effects, implemented persistent data storage with localStorage, and manipulated DOM dynamically with JavaScript.

⚠️ Challenges & Solutions

  • Persistent Cart: Solved using localStorage API.
  • Dynamic UI: Used JavaScript functions for real-time UI updates.

🧪 Testing Process & Features

Performed manual UI tests, ensured persistence, handled edge cases, and confirmed cross-browser compatibility.

  • Dynamic menu display with images and prices
  • Add to Cart, Order Summary, Clear Cart
  • localStorage persistence
  • Hover effects and transitions with Tailwind

🎯 Stretch Goal: Deep Dive into Tailwind

Explored utility classes deeply, used Tailwind documentation, practiced hover effects and layout utilities. No custom CSS was written, promoting skill development with Tailwind.

📑 Summary

The FoodZone app is a fully browser-based ordering tool with clean UI and strong use of Tailwind CSS. Learned DOM manipulation, localStorage, and utility-first CSS methodology through practical implementation.
