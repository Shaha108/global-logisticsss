FoodZone Restaurant Ordering App Documentation

üìù Project Overview

The FoodZone app is a client-side restaurant ordering interface built with plain HTML, JavaScript, and styled using Tailwind CSS. It allows users to browse a curated menu of food items, add them to a cart, and view a summary of their selections. The project emphasizes simplicity, focusing on dynamic interactivity and data persistence entirely within the browser.

Key parts of the code include:

  • Dynamic Menu Rendering: The menu items are rendered dynamically by JavaScript manipulating the DOM based on a predefined list of products.
  • Cart Management: Users can add items to a cart with quantities. The cart is stored and managed using the browser's localStorage to maintain data persistence between page reloads.
  • Order Summary & Controls: The app provides real-time updates on the cart contents, with controls to increment, decrement, or remove items.
  • UI Components: Tailwind CSS is used extensively to style components, including buttons, modals, cards, and interactive elements.

üéØ Stretch Goal: Deep Tailwind CSS Learning During the Project

How We Did a Deep Dive

  • Exploring the Utility Classes: Instead of writing custom CSS, every style was applied using Tailwind's utility classes, from spacing (p-4, m-2) to typography (font-semibold, text-gray-700).
