üìù Project Overview

The FoodZone app is a client-side restaurant ordering interface built with plain HTML, JavaScript, and styled using Tailwind CSS. It allows users to browse a curated menu of food items, add them to a cart, and view a summary of their selections. The project emphasizes simplicity, focusing on dynamic interactivity and data persistence entirely within the browser.

  • Dynamic Menu Rendering: Rendered by JavaScript manipulating the DOM based on a predefined list of products.
  • Cart Management: Cart stored using browser‚Äôs localStorage to maintain persistence.
  • Order Summary & Controls: Real-time cart updates with quantity controls.
  • UI Components: Styled with Tailwind CSS - buttons, modals, cards, etc.

üõ† Tools & Technologies Used

  • HTML5 & JavaScript: For structure and app logic.
